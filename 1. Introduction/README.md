# Introduction

## Hello World Program

> hello_world.cpp

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
}
```

---

## Compile the program

- `a.out` is created by default

```sh
g++ hello_world.cpp
```

```sh
g++ hello_world.cpp -o hello_world.o

g++ hello_world.cpp -o hello_world.out
```

---

## Run the Object file

```sh
./hello_world.o

./hello_world.out
```
